# Information Retrieval
Exercises for the course Information Retrieval given at the University of Amsterdam.  Labs in collaboration with [David Biertimpel](https://github.com/d4vidbiertmpl) and [Vanessa Botha](https://github.com/VanessaBotha).

Topics were 
- Query Processing
- Term-based Retrieval
- Offline Retrieval
- Semantic Retrieval
- Offline Learning-to-rank
- Online Learning-to-rank
- Knowledge graphs for text-centric retrieval
- Conversational search
